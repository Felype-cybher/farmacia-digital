# CAPS Gestão 💊

O **CAPS Gestão** é um sistema especializado para controle de estoque e distribuição de medicamentos em Centros de Atenção Psicossocial. Ele foi desenvolvido como **Projeto de Extensão** pelos alunos do curso de **Análise e Desenvolvimento de Sistemas (ADS)** do **IFMA - Campus Coelho Neto**.

O principal objetivo do projeto é modernizar e organizar o atendimento e a entrega de remédios no CAPS, facilitando o trabalho dos funcionários e garantindo o controle correto do estoque.

##  Divisão do Sistema

Para atender tanto a comunidade quanto os profissionais da saúde, o sistema é dividido em duas partes principais:

###  1. Área Pública (Acesso Livre)
* **Consulta Pública de Remédios:** Uma página aberta para qualquer cidadão pesquisar, em tempo real, se o medicamento que ele precisa está disponível no CAPS, sem precisar sair de casa.
* **Transparência:** Facilita o acesso à informação e evita que o paciente perca a viagem até a unidade.

###  2. Área Restrita (Acesso para Funcionários)
* **Painel de Controle (Dashboard):** Mostra gráficos rápidos com o total de remédios guardados, alertas do que está acabando e quais lotes estão perto do vencimento.
* **Histórico de Entradas e Saídas:** Registra cada vez que um medicamento chega ou é entregue para um paciente, atualizando a quantidade do estoque na hora.
* **Controle de Validade:** O sistema possui travas que impedem o cadastro de remédios vencidos, ajudando a evitar o desperdício.

---

## Segurança e Organização

* **Proteção de Dados:** A busca por pacientes e históricos traz apenas as informações de quem foi pesquisado na barra de busca. Isso protege a privacidade das pessoas e deixa o sistema mais rápido.
* **Bloqueio de Erros:** O sistema limpa caracteres estranhos na pesquisa para evitar que a tela quebre ou dê falhas.
* **Separação por Unidades:** Cada usuário logado só consegue ver e mexer nos dados da sua própria unidade do CAPS, garantindo total organização entre os setores.

## Tecnologias Utilizadas

* **Visual do Sistema (Frontend):** React.js, TypeScript, Tailwind CSS e Vite.
* **Banco de Dados (Backend):** Supabase (PostgreSQL).
